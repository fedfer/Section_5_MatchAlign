MSGP prior

We now estimate a Bayesian factor model using the MSGP prior, with with \(k\) latent variables.

We modified the function \(\texttt{linear_MSGP}\) contained in the \(\texttt{infinitefactor}\) package in order to deal with missing data. The new function \(\texttt{linear_MSGP_NA}\) is contained in the file \(\texttt{helper_fcts.R}\).

Missing data inputation is straightforward in Bayesian factor model. Given the values of \(\Lambda\), \(\{\eta_i\}_{i=1}^n\), and \(\Sigma=\text{diag}(\sigma_1^2,\dots,\sigma_p^2)\) at each iteration of the MCMC chain, any missing entry \(X_{ij}\) is sampled as \[ X_{i j} \sim N(\lambda_j^\top \eta_i, \sigma_j^2) \] where \(\lambda_j\) is the \(j^{th}\) row of \(\Lambda\).

out <- linearMGSP_NA(log(df) %>% as.matrix() %>% scale(), verbose = F,
                     nrun = 10000, burn = 5000, thin = 5, 
                     kinit = k, adapt = F, output = "factSamples")
lambdaSamps <- out$lambdaSamps
etaSamps <- out$etaSamps

lambda_fact_switch <- to_fact_switching(lambdaSamps)

Accuracy Metric

We further compute the accuracy metric (3.1): \[ || \overline{\Lambda\Lambda^T \!\!}\,\, - \bar \Lambda_* \bar \Lambda_* ^T ||_F \] where \(\overline{\Lambda\Lambda^T \!\!}\,\, = \frac{1}{T}\sum_{t = 1}^T \Lambda^{(t)}(\Lambda^{(t)})^T\) and \(\bar \Lambda_* = \frac{1}{T} \sum_{t = 1}^T \Lambda^{(t)}_*\) is the posterior mean of the aligned \(\Lambda\).

In order to give a more direct intuition of the effective benefit of each methodology, the metric values are expressed also as percentages of the relative error in the posterior mean of the covariance matrix \[ \frac{ || \overline{\Lambda\Lambda^T \!\!}\,\, - \bar \Lambda_* \bar \Lambda_*^T ||_F }{|| \overline{\Lambda\Lambda^T \!\!}\,\,\, ||_F} \]

# samples
Lambda_LambdaT <- lapply(lambdaSamps, function(x) x %*% t(x))
Lambda_LambdaT_mean <- lmean(Lambda_LambdaT)

# Base line 
base_norm <- norm(Lambda_LambdaT_mean, type = "F")

# WOP
Align_cov <- lmean(Lambda_WOP)
Align_cov_mean <- Align_cov %*% t(Align_cov)
WOP_norm <- norm(Align_cov_mean - Lambda_LambdaT_mean, type = "F")

# RSP-full-SA
Align_cov <- lmean(Lambda_rspFull)
Align_cov_mean <- Align_cov %*% t(Align_cov)

# RSP-partial-SA

Align_cov_mean <- Align_cov %*% t(Align_cov)
